tricky -LEVEL QUESTIONS (WITH ANSWERS)

❓ Q1
console.log([] + []);
✅ Answer:
""

➡ [] → ""
➡ "" + "" → ""


❓ Q2
console.log([] + {});
✅ Answer:
"[object Object]"



❓ Q3
console.log({} + []);
✅ Answer:
0
❗ Because {} is treated as a block, not object


❓ Q4
console.log("5" * "2");
✅ Answer:
10
✔ * forces numeric conversion


❓ Q5
console.log(+"1" + 2 + "2");
✅ Output:
32
(+"1" converts string → number)
